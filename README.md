# Coffee-Shop-Sales-Dashboard
## Project Overview 
This project analyzes coffee shop sales data using PostgreSQL and Power BI to generate actionable business insights. It includes revenue analysis, transaction tracking, monthly sales trends, top-selling products, and customer purchasing patterns. Interactive dashboards and KPIs help evaluate performance and support data-driven decision-making.

## Tools Used
- PostgreSQL
- Power BI
- SQL
- DAX

## Key Metrics
- Total Revenue
- Total Transactions
- Average Daily Revenue
- Best Sales Day
- Top-Selling Products
- Monthly Sales Trends

## Dashboard Features
- Revenue Analysis
- Product Performance Analysis
- Transaction Insights
- Monthly Sales Comparison
- Interactive Filters and Slicers

## Business Insights
- Identified highest revenue-generating products.
- Analyzed peak sales periods and best-performing days.
- Evaluated monthly sales growth trends.
- Tracked transaction volume and customer purchasing behavior.
